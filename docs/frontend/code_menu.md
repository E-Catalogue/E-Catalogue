# List Code Menu & Permissions

Berikut adalah daftar `code` permission yang digunakan dalam sistem (berdasarkan modul). Code ini diatur di tabel permissions dan dikirimkan pada saat login/refresh (`permissionCodes`) untuk keperluan validasi otoritas UI di Front-End dan Back-End.

## Modul Inti & Akses
- **Auth**: (Tidak ada permission khusus, endpoint `/auth/me` & `/auth/logout` hanya mengecek validitas token)
- **Role**: `ROLE_READ`, `ROLE_CREATE`, `ROLE_UPDATE`, `ROLE_DELETE`, `ROLE_SET_PERMISSION`
- **User**: `USER_READ`, `USER_CREATE`, `USER_UPDATE`, `USER_DELETE`, `USER_SET_ROLE`, `USER_SET_BRANCH`
- **Menu**: `MENU_READ`, `MENU_CREATE`, `MENU_UPDATE`, `MENU_DELETE`, `PERMISSION_MANAGE`
